⚙️ Bulk Operations

Here you can make bulk operations like exporting, importing and deleting data. You can also outsource the creation of scheduled posts to other freelancers and import them in the app. First download the excel template which the freelancer (or you) can fill with the posts by clicking 'Download excel template for scheduled posts' button. The excel file will contain the following columns: "Post at YYYY-MM-DD at HH:MM" (here you have to put the date like "2024-01-23 at 17:20"), "Title", "Description", "File Name" (here you have to put the filename from the folder 'files'), "X", "Instagram", "Facebook", "Tiktok", "Linkedin", "Youtube" (put 'yes'/'y' or 'no'/'n' so the app knows where to post it), Next to that excel file you need to create a folder named 'files' in which you put the related files. Now you can select the folder and the excel file and make a zip from them which can be imported in the app by clicking 'Import .zip file with scheduled posts'. If dates are in the past - the posts will be automatically moved in the future.
