Privacy Policy

Data Collection: imposting.com collects the following user data: username, name, and profile picture, if the user grants access to imposting.com to post via API. This information is displayed when the user clicks the Authorize button on the Integrations page in the modal corresponding to one of the platforms supported by imposting.com for scheduling posts.

Data Use and Processing: The collected data is saved to a database via SSL and used on the Integrations page to display the username and profile picture next to the integration authorized by the user. This helps users identify which pages/profiles are managed by imposting.com. Currently, only one account is supported for each platform (e.g., one Facebook public page, one Instagram business account, one X account, one LinkedIn account, one TikTok business account). Login with Google is used to authenticate users via OAuth2. All data is used to enable imposting.com to post via API on behalf of the user and to identify users in case they require support.

Data Storage and Security: User data is stored in a cloud-based SQL database. HTTPS/SSL is used to secure data during transfer. The application packages and operating system are regularly updated to ensure the app remains secure from potential threats. While no system is entirely immune to breaches, we continuously monitor the application's health and take action as needed.

User Rights: Users can access their data within the app and delete it or revoke access to external APIs on the Integration page. Users can also revoke access on the respective third-party apps. For example, users can forbid access to imposting.com by adjusting their Google settings. Users can request their data to be sent to them via email by contacting alinclimente@softgata.com.

Data Sharing: User data is not shared with third parties.

Cookie Policy

Cookie Usage: Cookies are stored in the browser to provide secure authentication for users and to protect forms with CSRF tokens. Local storage is used to store user preferences, such as cookie consent, to enhance the user experience.

Managing Cookie Preferences: Since cookies are essential for authentication, deleting them from the browser will require users to re-authenticate for each request made to imposting.com.
